An OBJ viewer written in go
===================

Requirements:

Go Weekly

Install [gogl](http://github.com/chsc/gogl):

```bash
go get 'github.com/chsc/gogl'
```

Installation
----------------

```bash
git clone git://github.com/Fumon/go-obj-view.git
cd go-obj-view
./build.sh
```

Running
-----------

```bash
cd bin
./monkey -file="somefile.obj"
```
